Bubble Sort

Pseudocode

do

  swapped = false

  for(0 to lastUnsortedElement - 1)

    if(elements[i] > elements[i + 1])

      swap(elements[i], elements[i + 1])

      swapped = true

while(swapped = true)

Description: Bubble Sort is a simple algorithm that repeatedly iterates through elements, swapping adjacent elements if they are in the wrong order.

Average Time Complexity: O(n2)

Best Time Complexity: O(n) when the list is already sorted, or 1 element is at an index before its sorted location.

Worst Time Complexity: O(n2)

Space Complexity: O(1)

Merge Sort

Pseudocode

function mergeSort(input[])

  if(inputLength = 1)

    return input

  else

    lSub[] = mergeSort(input[0 to mid])

    rSub[] = mergeSort(input[mid to end]

    for(0 to inputLength)

      if(lSub[0] < rSub[0])

        add lSub[0] to output

      else

        add rSub[0] to output

    return output

Description: Merge Sort is a divide and conquer algorithm that divides an array into 2 halves, calls itself on each half, then merges the halves back together.

Time Complexity: O(nlogn)

Space Complexity: O(n)

Quick Sort

Pseudocode

function quickSort(start, end)

  if(end - start > 1)

    pivot = input[start]

    swapIndex = start + 1

    for(start + 1 to end)

      if(currentElement < pivot)

        swap(currentElement, swapIndex)

        swapIndex++;

    swap(start, swapIndex - 1)

    quickSort(start, swapIndex - 1)

    quickSort(swapIndex, end)

Description: Quick Sort is a divide and conquer algorithm that picks a pivot element and puts it at the correct position by putting all elements less than it before it. This is repeated until the array is sorted.

Average Time Complexity: O(nlogn)

Best Time Complexity: O(nlogn) when the selected pivot is always the median element.

Worst Time Complexity: O(n2) when the selected pivot is always the largest or smallest unsorted element.

Space Complexity: O(logn)

Radix Sort

Pseudocode

create queues for each radix

for each significant digit

  for each element in input[]

    sort element into respective queue

  for each queue

    while queue length > 0

      push first queue element to input

Description: Radix Sort is a non-comparative sorting algorithm that sorts elements by radix by grouping elements if they share the same digit at the same significant digit position. This process is repeated for each digit from least significant to most significant digit.

Time Complexity: O(d*(n+b)) where b is the base for representing numbers, and d is logb(maxValue). Put simply, b is the number of radixes (in the decimal system this would be 10), and d is the length of the longest element.

Space Complexity: O(n+b) where b is the base for representing numbers.

Selection Sort

Pseudocode

for(0 to input[] length - 1)

  minimum = input[0]

  for each unsorted element

    if(element < minimum)

      minimum = element

  swap(minimum, first unsorted element)

Description: Selection Sort is a simple sorting algorithm that repeatedly finds the minimum element, and swaps it with the first unsorted element, and marks it as sorted

Time Complexity: O(n2)

Space Complexity: O(1)
